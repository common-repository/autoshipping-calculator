== Description ==

This plugin allows you to easily add a widget to your sidebar or post content which makes your visitors able to instantly get a quote on auto transport services.
The options you will have avaiable are choosing from predefined sizes that fit in the best way the calculator itself. You will be able to choose from custom sizes as you need.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `autoshipping-calculator.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add and configure the widget
4. Adjust the plugin settings in order to add the calculator to your posts
